## Strategies to Maximize Free Tier

### 1. Use Multiple Search Queries

Instead of one broad search, break it down:

**Example: Looking for marketing managers in San Francisco**

Run separate searches:
- Search 1: "Marketing Manager" + Company: "Salesforce"
- Search 2: "Marketing Manager" + Company: "Airbnb"
- Search 3: "Marketing Manager" + Company: "Stripe"
- Search 4: "Marketing Manager" + School: "Stanford University"

**Why this works:**
- Each search shows ~1000 results
- 10 targeted searches = 10,000 potential prospects
- More targeted = better quality connections

### 2. Leverage My Network Page

**Best practices:**
1. Go to linkedin.com/mynetwork/
2. LinkedIn shows personalized recommendations
3. Scan these - they're often high-quality matches
4. Refresh the page after a few days for new suggestions

**Why this works:**
- LinkedIn's algorithm shows relevant people
- Usually 2nd degree connections
- High acceptance rate (LinkedIn recommends them)

### 3. Search by Company/School

**Strategy:**
```
1. Identify 20 target companies
2. Search: People who work at [Company Name]
3. Scan each company separately
4. Repeat for schools/universities
```

**Why this works:**
- Bypasses search result limits
- More specific = better targeting
- Can collect thousands of prospects

### 4. Use Boolean Searches

LinkedIn free tier supports Boolean operators:

**Examples:**
- `"Software Engineer" OR "Developer"` - finds either title
- `"CEO" AND "startup"` - finds CEOs mentioning startups
- `Marketing NOT Manager` - excludes managers

**Why this works:**
- Refines results without Premium filters
- Gets around free tier limitations
- More precise targeting

### 5. Geographic Filtering

**Strategy:**
```
Search in specific cities one at a time:
- San Francisco
- New York
- Austin
- Remote
```

**Why this works:**
- Each location = separate 1000 result pool
- Better localized targeting
- Easier to personalize connection requests

## Tips to Get More Value on Free Tier

Before scanning, make sure your LinkedIn profile is strong:

Priority order:

- Track who you've contacted in a spreadsheet
